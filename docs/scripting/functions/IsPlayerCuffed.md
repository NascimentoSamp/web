## Description

Check if the player special action is cuffed.

| Name     | Description                                                 |
| -------- | ----------------------------------------------------------- |
| playerid | The ID of the player to check.       |

## Returns

Returns true if the player is cuffed, otherwise false.

## Examples

```c
if (IsPlayerCuffed(playerid))
{
    // do something
}
```
